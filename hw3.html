Problem 1¶

На сайте PubChem найти все радикалы c азидом для Click Chemistry и скачать их SMILES нотации.

Wikipedia:
Азиды — органические соединения, содержащие азидную группу $\require{mhchem}\ce{-N=N+=N^-}$, обычно связанную с атомом углерода. Мезомерия:

Имеем следующие smiles(на самом деле smarts) для substructure: N=[N+]=[N-;!R], [N-]-[N+]#[N;!R], [N-]-N=[N+;!R].

Используем паттерн !R, чтобы выбрать только отдельные азидные группы вне кольца.

Problem 5¶

Отобрать те молекулы, которые удовлетворяют правилу пяти Lipinski

In [22]:
newcandidates = []

for mol in candidates:
    hits = 0
    
    if Lipinksy.NumHDonors(mol) <= 5:
        hits += 1
    if Lipinksy.NumHAcceptors(mol) <= 10:
        hits += 1
    if Lipinksy.rdMolDescriptors.CalcExactMolWt(mol) <= 500:
        hits += 1
    if Lipinksy.rdMolDescriptors.CalcCrippenDescriptors(mol)[0] <= 5:
        hits += 1
    if hits >= 3:
        newcandidates.append(mol)

candidates = newcandidates
